Common HTML Tags Reference

1. Basic Structure Tags

  • <html>: The root element that wraps all content on the page.
  • <head>: Contains metadata and links to stylesheets, scripts, and other resources.
  • <title>: Defines the title of the document (displayed on the browser tab).
  • <body>: Contains the visible content of the webpage.

2. Text Content Tags

  • <h1> to <h6>: Heading tags, <h1> is the largest, <h6> is the smallest.
  • <p>: Paragraph tag for blocks of text.
  • <a>: Anchor tag for hyperlinks. Example: 
    <a href="https://example.com">Click Here</a>
  • <strong>: Indicates strong importance, typically rendered in bold.
  • <em>: Emphasizes text, typically rendered in italics.
  • <br>: Inserts a line break.
  • <hr>: Creates a horizontal line (thematic break).
  • <span>: Inline container for styling or grouping small parts of text.
  • <div>: Block-level container for grouping content.

3. List Tags

  • <ul>: Unordered list (bulleted).
  • <ol>: Ordered list (numbered).
  • <dl>: Definition list.

4. Media Tags

  • <img>: Displays an image. Example: 
    <img src="image.jpg" alt="Description">
  • <audio>: Embeds audio files. Example: 
    <audio controls src="audio.mp3"></audio>
  • <video>: Embeds video files. Example: 
    <video controls src="video.mp4"></video>
  • <iframe>: Embeds external content. Example: 
    <iframe src="https://example.com"></iframe>

5. Table Tags

  • <table>: Defines a table.
  • <tr>: Table row.
  • <th>: Table header cell.
  • <td>: Table data cell.
  • <thead>: Groups header rows.
  • <tbody>: Groups body rows.
  • <tfoot>: Groups footer rows.

6. Form Tags

  • <form>: Creates a form for user input.
  • <input>: Accepts user input. Example: 
    <input type="text" placeholder="Enter your name">
  • <label>: Labels for form elements (improves accessibility).
  • <textarea>: Multiline text input.
  • <button>: Clickable button.
  • <select>: Dropdown menu.

7. Semantic Tags

  • <header>: Represents a header section.
  • <nav>: Defines navigation links.
  • <main>: Main content of the page.
  • <section>: Groups related content.
  • <article>: Represents an independent piece of content.
  • <aside>: Sidebar content, such as ads or related links.
  • <footer>: Footer section of a document.

8. Metadata and Resource Tags

  • <meta>: Defines metadata for the document (e.g., charset, viewport).
  • <link>: Links external resources, like stylesheets. Example: 
    <link rel="stylesheet" href="styles.css">
  • <style>: Embeds CSS styles directly in HTML.
  • <script>: Embeds JavaScript code or links an external JS file.

9. Interactive Content Tags

  • <details>: Creates a collapsible content block.
  • <dialog>: Defines a dialog box or popup.
  • <progress>: Displays progress of a task.
  • <meter>: Represents a scalar measurement (e.g., disk usage).

10. Other Useful Tags

  • <abbr>: Represents an abbreviation or acronym. Example: 
    <abbr title="HyperText Markup Language">HTML</abbr>
  • <code>: Displays code snippets.
  • <pre>: Preserves whitespace formatting.
  • <blockquote>: For long quotations.
  • <cite>: Cites a source.
  • <mark>: Highlights text.
  • <time>: Represents a time or date.
  • <address>: Represents contact information. Example: 
    <address>123 Main Street, Anytown, USA</address>
